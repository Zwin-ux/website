# Bonelli Labs

Marketing site for two Bonelli Labs products:

- Synergy / ScriptLens
- Superior

Built with Next.js, React, TypeScript, and Tailwind CSS.

## Routes

- `/`
- `/synergy`
- `/superior`

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

## Deploy

Railway can deploy this as a standard Next.js app from Git with no app-specific config required.
